#arch-bootstrap-thinkpad
Instructions for bootstrapping the configuration of Arch Linux on a 3rd Generation Lenovo Thinkpad Caron X1. 

1. Follow instructions in the [Arch Beginner's Guide](https://wiki.archlinux.org/index.php/Beginners%27_guide).
2. `source <(curl -s https://raw.githubusercontent.com/evandbrown/arch-bootstrap-thinkpad/master/bootstrap.sh)`
